# Changelog

## Unreleased

### Added

- Production CRUD with search, client/server pagination, sorting, details, batch deletion and row-level permissions.
- Element Plus confirmation, success and error feedback for CRUD operations.
- Custom field registry with plugin-level registration.
- Conditional visibility and disabled state, dependent fields, computed values and asynchronous options.
- Custom synchronous and asynchronous field validation.
- Paginated API responses, request timeout, retry and request/response hooks.
- Safer SFC generation for special field names and untrusted labels.
- Code-generation warnings for runtime functions that cannot be serialized.
